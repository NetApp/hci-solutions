Details on Heat Templates

Provided are heat orchestration templates (HOT), instack.json, and basic deployment script for the deployment and configuration of the Red Hat OpenStack Platform OverCloud, as detailed in NetApp NVA-1133: NetApp HCI for Private Cloud with Red Hat.

The templates provided were used with the validated deployment as detailed in the NVA, providing a basic OSP cloud environment as well as some additional features such as enabling volumes mapped from NetApp Element to be used for the backend of Swift object storage and configuration of the Octavia load-balancer used for OpenShift. 

Please consult with your NetApp professional services team in order to modify these templates for use in your specific private cloud environment.
